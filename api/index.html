    pyfgs API Reference¶

    This section provides the complete API documentation for pyfgs, parsed directly from the type stubs.

    Main Classes¶

    pyfgs.GeneFinder ¶

    The main engine for finding genes, holding the HMM in memory.

    Example 
    from pyfgs import GeneFinder, Model

# For complete genomes or error-free reads
finder = GeneFinder(Model.Complete)

# For short, error-prone reads (e.g., Illumina with 0.5% error)
finder_short_reads = GeneFinder(Model.Illumina5)

    Initialize the GeneFinder.

    Parameters:

    Name Type Description Default
    model Model

    The sequencing error model to use.

    		 required
    whole_genome Optional[bool]

    Set to True if analyzing complete genomic sequences. If None, it defaults to True for Model.Complete, and False for all short-read models.

    find_genes(header, sequence) ¶

    Predict open reading frames in a given DNA sequence.

    This method releases the GIL, allowing for safe multi-threading across multiple CPU cores.

    Parameters:

    Name Type Description Default
    header bytes

    The sequence identifier.

    		 required
    sequence bytes

    The raw nucleotide buffer.

    		 required

    Returns:

    Type Description
    List[Gene]

    List[Gene]: A list of predicted Gene objects.
    Example 
    from pyfgs import GeneFinder, Model

finder = GeneFinder(Model.Complete)
header = b"seq1"
sequence = b"ATGCGTACGTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTA"

genes = finder.find_genes(header, sequence)
for gene in genes:
    print(f"Found gene from {gene.start} to {gene.end} on strand {gene.strand}")
    # Lazily get the translated protein sequence
    protein = gene.translation()
    print(f"Protein: {protein.decode()}")

    pyfgs.Gene ¶

    Represents a single predicted Open Reading Frame (ORF).

    Gene objects are not created directly, but are returned by GeneFinder.find_genes.

    Example 
    # Assuming `gene` is a Gene object from a GeneFinder
print(f"Found gene: start={gene.start}, end={gene.end}, strand={gene.strand}")

# Lazily get the DNA and protein sequence as bytes
dna_seq = gene.sequence()
protein_seq = gene.translation()

print(f"DNA: {dna_seq.decode()}")
print(f"Protein: {protein_seq.decode()}")

    deletions property ¶

    A list of 0-based indices representing deletion sequencing errors. These are positions where a gap was inserted to maintain the reading frame.

    end property ¶

    The 0-based, half-open end coordinate of the ORF.

    frame property ¶

    int: The reading frame (1, 2, or 3).

    insertions property ¶

    A list of 0-based indices representing insertion sequencing errors. These are bases in the original sequence that were skipped to maintain the reading frame.

    score property ¶

    float: The Viterbi score of the gene prediction.

    start property ¶

    The 0-based start coordinate of the ORF.

    strand property ¶

    The strand of the ORF (1 for forward, -1 for reverse).

    sequence() ¶

    Lazily compute the raw nucleotide sequence of the ORF.

    translation() ¶

    Lazily compute the translated amino acid sequence of the ORF.

    Parsers¶

    pyfgs.FastaReader ¶

    A memory-efficient FASTA parser yielding (header, sequence).

    The header and sequence are yielded as raw bytes, which avoids any UTF-8 decoding overhead.

    Example 
    from pyfgs import FastaReader

# Assuming "genome.fasta" contains:
# >seq1
# ATGCGT

reader = FastaReader("genome.fasta")
for header, sequence in reader:
    # header: b'seq1', sequence: b'ATGCGT'
    print(f">{header.decode()}\n{sequence.decode()}")

    pyfgs.FastqReader ¶

    A memory-efficient FASTQ parser yielding (header, sequence, qualities).

    The header, sequence, and quality scores are yielded as raw bytes, which avoids any UTF-8 decoding overhead.

    Example 
    from pyfgs import FastqReader

# Assuming "reads.fastq" contains:
# @read1
# ATGCGT
# +
# IIIIII

reader = FastqReader("reads.fastq")
for header, sequence, qualities in reader:
    # header: b'read1', sequence: b'ATGCGT', qualities: b'IIIIII'
    print(f"@{header.decode()}\n{sequence.decode()}\n+\n{qualities.decode()}")
